# Purpose

This is a 3D model of a chassis designed to hold:
* the water pump
* main PCB

# Tools

The model was made in OpenSCAD version 2021.01 and then exported to FreeCAD version 1.0.

# Usage

* Model the chassis in OpenScad
* Open FreeCad, create new body, File -> Import, select Main.scad
* Add fillets
* Export for 3d printing
